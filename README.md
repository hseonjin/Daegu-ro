### 구현 기능

#### 1. DashBoard

- 네이버 블로그를 통해 데이터를 크롤링하여 관광수요를 분석
- 분석한 데이터를 그래프로 시각화하여 웹 페이지에 노출

#### 2. Recommend

- 10가지 테마에 맞게 여행지 카테고리화
- 카테고리별 여행지 리스트, 상세버튼 클릭 시 modal을 통해 여행지 소개
- kakaomap API를 통해 카테고리별 전체 여행지 marker
